# 02 - UI Components (Storefront)

Thư viện component đặc thù của giao diện cửa hàng PI.

## 1. Homepage Sections
- **HomeHero**: Khu vực giới thiệu chính với hiệu ứng "Orbit" chuyển động 3D.
- **HomeBento**: Hệ thống lưới Bento Grid giới thiệu các tính năng AI cốt lõi.
- **FeaturedProducts**: Slider hoặc Grid hiển thị các sản phẩm nổi bật nhất.
- **HomeCTA**: Các khối kêu gọi hành động với hiệu ứng phát sáng Quantum HUD.

## 2. Product Components
- **ProductCard**: Thẻ hiển thị sản phẩm với thông tin giá, badge ưu đãi và nút mua nhanh.
- **PricingTable**: Bảng so sánh tính năng giữa các gói Free/Pro/Max.
- **FeatureList**: Danh sách các tính năng đi kèm của từng plugin.

## 3. Checkout & Cart
- **CartDrawer**: Giỏ hàng trượt từ cạnh phải, cho phép xem nhanh và điều chỉnh số lượng.
- **CheckoutForm**: Form thanh toán tối giản, tích hợp Stripe Elements và kiểm tra License.
- **OrderSummary**: Bảng tổng hợp đơn hàng, thuế và tổng tiền.

## 4. UI Primitives (Atomic)
- **GlassShell**: Wrapper tạo hiệu ứng kính mờ cho các card.
- **Badge**: Các nhãn đánh dấu sản phẩm (New, Hot, Pro).
- **Button**: Các nút bấm với hiệu ứng hover nâng cao và animation khi click.
